<h1 align=center>Cheat Sheet</h1>

Primitive Types:
  + INTEGER:
    + `byte` (8bit),
    + `short` (16bit),
    + `int` (32bit),
    + `long` (64bit),
  + DECIM:
    + `float` (32bit),
    + `double` (64bit),
  + OTHER:
    + `boolean` (1bit),
    + `char` (Unicode)
  + HEX:
    + 0x0AF,
  + BINARY:
    + 0b01101,
  + LONG:
    + 1234567890123L
  + CHAR EXAMPLES: `a`, `\n`, `\t`, `\’`, `\\`, `\”`
