<h1 align=center>Cheat Sheet</h1>

Primitive Types:
  + INTEGER:
    + `byte` (8bit),
    + `short` (16bit),
    + `int` (32bit),
    + `long` (64bit),
  + DECIM:
    + `float` (32bit),
    + `double` (64bit),
  + OTHER:
    + `boolean` (1bit),
    + `char` (Unicode)
  + HEX:
    + 0x1AF,
  + BINARY:
    + 0b00101,
  + LONG:
    + 8888888888888L
        
  + CHAR EXAMPLES: 
     `a`,
     `\n`,
     `\t`,
     `\’`,
     `\\`,
     `\”`
